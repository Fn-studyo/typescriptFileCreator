# Creator



Creator is a tool that helps you create a basic route, controller and model for a simple typescript app


## Usage

Creator requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and start the server.

```sh
$ npm i
$ npm start
$ node creator.js
```
